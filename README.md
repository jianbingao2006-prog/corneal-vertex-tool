# 角膜顶点偏移量工具

医疗工具：根据角膜 OS / OD 直角坐标输入，自动计算极坐标并生成可打印 / 导出的高清示意图。

## 在线使用

打开 [index.html](index.html) 即可使用。

## 安卓 APK 安装包

每次 push 到 main 分支会自动通过 GitHub Actions 构建 APK：

1. 进入仓库 **Actions** 标签页
2. 点最新一次的 "Build APK" 运行
3. 滚到底部，在 **Artifacts** 区域下载 `corneal-vertex-tool-apk.zip`
4. 解压后里面的 `app-release-signed.apk` 就是要装到手机的安装包

APK 安装在 vivo / 小米 / 华为 / OPPO 等所有安卓 5.0 以上手机均可使用。
